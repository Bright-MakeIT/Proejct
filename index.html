AutoEncoder 기반 이상 거래 탐지

Python & PyTorch로 구현한 비지도 학습 AI 프로젝트

📌 프로젝트 개요

Deep Learning Anomaly Detection PyTorch

정상 거래 패턴을 학습한 뒤 재구성 오차(Reconstruction Error)를 기반으로 이상 거래를 탐지하는 AutoEncoder 모델을 구현했습니다.

🧠 모델 구조

Input(30) → Encoder(16 → 8) → Bottleneck → Decoder(8 → 16) → Output(30)

Loss Function: MSE (Mean Squared Error)
Optimizer: Adam

💻 Google Colab 실습 코드

아래 코드를 복사하여 Google Colab에서 실행 가능합니다.


                        # 라이브러리 설치
                        !pip install torch pandas scikit-learn
                        
                        import torch
                        import torch.nn as nn
                        import pandas as pd
                        import numpy as np
                        from sklearn.preprocessing import StandardScaler
                        from sklearn.metrics import classification_report
                        
                        # 데이터 업로드
                        from google.colab import files
                        uploaded = files.upload()
                        filename = list(uploaded.keys())[0]
                        df = pd.read_csv(filename)
                        
                        X = df.drop(['Class'], axis=1)
                        y = df['Class']
                        
                        scaler = StandardScaler()
                        X_scaled = scaler.fit_transform(X)
                        X_tensor = torch.FloatTensor(X_scaled)
                        
                        X_train = X_tensor[y == 0]
                        
                        class AutoEncoder(nn.Module):
                            def __init__(self):
                                super(AutoEncoder, self).__init__()
                                self.encoder = nn.Sequential(
                                    nn.Linear(30,16),
                                    nn.ReLU(),
                                    nn.Linear(16,8)
                                )
                                self.decoder = nn.Sequential(
                                    nn.Linear(8,16),
                                    nn.ReLU(),
                                    nn.Linear(16,30)
                                )
                            def forward(self,x):
                                encoded = self.encoder(x)
                                decoded = self.decoder(encoded)
                                return decoded
                        
                        model = AutoEncoder()
                        criterion = nn.MSELoss()
                        optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
                        
                        for epoch in range(20):
                            optimizer.zero_grad()
                            output = model(X_train)
                            loss = criterion(output, X_train)
                            loss.backward()
                            optimizer.step()
                            print(f"Epoch {epoch}, Loss: {loss.item()}")
                        
                        with torch.no_grad():
                            reconstructed = model(X_tensor)
                            errors = torch.mean((X_tensor - reconstructed)**2, dim=1)
                        
                        threshold = np.percentile(errors.numpy(), 95)
                        preds = (errors.numpy() > threshold).astype(int)
                        
                        print(classification_report(y, preds))

딥러닝 기반 MNIST 손글씨 인식

Python & TensorFlow/Keras 기반 이미지 분류

프로젝트 개요

TensorFlow 2.x CNN Accuracy 98%+

MNIST 데이터셋을 활용해 0~9 숫자를 인식하는 CNN(Convolutional Neural Network)을 구축했습니다.

모델 아키텍처


                        # 데이터 로드 & 시각화 (Google Colab 추천)
                        import tensorflow as tf
                        import matplotlib.pyplot as plt
                        
                        mnist = tf.keras.datasets.mnist
                        (x_train, y_train), (x_test, y_test) = mnist.load_data()
                        
                        x_train, x_test = x_train / 255.0, x_test / 255.0
                        
                        plt.figure(figsize=(10,10))
                        for i in range(25):
                            plt.subplot(5,5,i+1)
                            plt.xticks([]); plt.yticks([]); plt.grid(False)
                            plt.imshow(x_train[i], cmap=plt.cm.binary)
                            plt.xlabel(y_train[i])
                        plt.show()

→ Conv → Pooling → Conv → Pooling → Flatten → Dense → Softmax

💻 모델 훈련 & 평가 코드


                        # 데이터 reshape (CNN 입력 형식)
                        x_train = x_train[..., tf.newaxis]
                        x_test = x_test[..., tf.newaxis]
                        
                        model.compile(optimizer='adam',
                                      loss='sparse_categorical_crossentropy',
                                      metrics=['accuracy'])
                        
                        history = model.fit(x_train, y_train, epochs=5, validation_data=(x_test, y_test))
                        
                        test_loss, test_acc = model.evaluate(x_test, y_test, verbose=2)
                        print(f"Test accuracy: {test_acc:.4f}")
                        
                        # 학습 곡선
                        plt.plot(history.history['accuracy'], label='train')
                        plt.plot(history.history['val_accuracy'], label='val')
                        plt.xlabel('Epoch'); 
                        plt.ylabel('Accuracy')
                        plt.legend(); 
                        plt.show()

컴퓨터 비전: YOLOv8 기반 객체 탐지 프로젝트

Python & Ultralytics를 활용한 실시간 객체 인식 시스템 구축

프로젝트 개요

YOLOv8 PyTorch mAP 0.75+

이 프로젝트는 YOLOv8 모델을 사용해 이미지/비디오에서 객체를 탐지하고 분류하는 시스템을 구현합니다. COCO 데이터셋이나 커스텀 데이터(예: 교통 표지판)를 훈련해 실무 적용 가능성을 강조합니다. MNIST보다 훨씬 advanced하며, 자율주행이나 보안 시스템 같은 산업 프로젝트에 적합합니다. 자율주행 및 보안 분야 응용 가능성이 높습니다.

  • 목표: 실시간 탐지 속도 30FPS 이상, 평균 정밀도(mAP) 0.75 이상 달성
  • 주요 기술: Python, PyTorch, Ultralytics YOLO, OpenCV, NumPy
  • 학습 과정: 데이터 준비 → 모델 훈련 → 추론 → 성능 평가

                        cv2_imshow(img)

                        # Colab에서 실행: 모델 훈련 예시 (YAML 파일 가정)

                        # 훈련 결과 저장
